# Runnable examples

Scripts in `examples/` and how to run them:

| Script | What it does |
|--------|----------------|
| `npm run example:quick-start` | [run-quick-start.ts](../examples/run-quick-start.ts) - Super quick start: `createQuickAgent`, one goal, no steps. Needs `OPENAI_API_KEY`. |
| `npm run example:fluent-plan` | [run-fluent-plan.ts](../examples/run-fluent-plan.ts) - Fluent plan (`step()` + `buildPlan`) and default executor. Needs `OPENAI_API_KEY`. |
| `npm run example:goal-driven-llm` | [run-goal-driven-llm.ts](../examples/run-goal-driven-llm.ts) - Goal-only, steps generated by the LLM. Needs `OPENAI_API_KEY`. Optional goal: `npm run example:goal-driven-llm -- "Your goal"`. |
| `npm run example:minimal` | [run-minimal.ts](../examples/run-minimal.ts) - Minimal planner, executor, and evaluator (no API key, stub outputs). |
| `npm run example:openai` | [run-with-openai.ts](../examples/run-with-openai.ts) - Full run with tools (list dir, read file, grep) and OpenAI or local (Ollama/LM Studio). |
| `npm run example:planner` | [run-project-planner.ts](../examples/run-project-planner.ts) - Project structure from a short description. |
| `npm run example` | [run-example.ts](../examples/run-example.ts) - Stub planner/executor/evaluator, no API. |
